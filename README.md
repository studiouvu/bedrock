# About

![img](https://bedrock.md/images/screenshot-2.png)

ASP.NET으로 개발된 To-do 리스트 사이트입니다.  
간단하게 할 일을 정리하기 위해 사이드 프로젝트로 개발했습니다.

# Features

- 온라인 베이스로 동작하여 PC, Mobile 연동
- 할 일들을 AI로 정리 후 요약
- Email을 통한 로그인 기능
